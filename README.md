# دليل تشغيل لوحة التحكم "Youcan" محلياً على جهاز الكمبيوتر الخاص بك 💻

مرحباً بك! لقد قمنا بتهيئة وتطوير التطبيق ليعمل **محلياً على اللاب توب الخاص بك** مع الربط والمزامنة الكاملة مع **جدول بيانات جوجل (Google Sheets)**، وهو مصدر البيانات الرئيسي والنهائي للتطبيق.

يتم تخزين البيانات محلياً بشكل مؤقت في المجلد `data/db.json` لتجربة تصفح سريعة وخارقة السرعة، ويتم الاتصال والتكامل بشكل فوري ومباشر مع Google Sheets لقراءة وتحديث طلبات المبيعات، ومشتريات السلع، ودفعات الموردين، والمصاريف.

---

## 🛠️ المتطلبات الأساسية للتشغيل

لتشغيل التطبيق على جهاز الكمبيوتر الخاص بك، تحتاج فقط إلى تثبيت برنامج **Node.js** بروابطه وآمنه تماماً:

1. قم بتحميل وتثبيت **Node.js** من الموقع الرسمي: [https://nodejs.org](https://nodejs.org) (اختر نسخة **LTS** المستقرة والموصى بها لغالبية المستخدمين).
2. قم بتثبيت البرنامج بـ الإعدادات الافتراضية بالضغط على Next حتى النهاية.

---

## 🚀 خطوات التشغيل خطوة بخطوة

بعد تحميل كود التطبيق كـ ملف مضغوط (ZIP) من منصة AI Studio وفك الضغط عنه على قرصك الصلب:

### 1️⃣ فتح موجه الأوامر (Terminal / Command Prompt)
* **على نظام Windows:** اضغط على زر `Windows + R` على لوحة المفاتيح، ثم اكتب `cmd` واضغط Enter.
* **على نظام Mac:** افتح تطبيق **Terminal**.

### 2️⃣ الانتقال إلى مجلد التطبيق
اكتب أمر `cd` متبوعاً بمسار المجلد الذي قمت بفك الضغط فيه. على سبيل المثال:
```bash
cd "C:\Users\YourName\Desktop\youcan-dashboard"
```

### 3️⃣ تثبيت الحزم والمكتبات اللازمة (تُجرى أول مرة فقط)
اكتب الأمر التالي واضغط Enter لتقوم الأداة بتحميل جميع المكتبات الضرورية:
```bash
npm install
```
*انتظر دقيقة أو دقيقتين حتى تكتمل العملية.*

### 4️⃣ بدء تشغيل التطبيق محلياً
لتشغيل التطبيق واستخدامه، تشغل السيرفر المحلي بكتابة:
```bash
npm run dev
```

بمجرد ظهور رسالة النجاح في السيرفر:
```text
Server running on http://localhost:3000
```

انتقل إلى متصفح الإنترنت المفضل لديك وافتح هذا الرابط:
👉 **[http://localhost:3000](http://localhost:3000)**

## 📊 كيفية الإعداد والربط مع Google Sheets (مصدر البيانات)

لضمان سحب البيانات وعرضها بشكل كامل، اتبع الخطوات البسيطة التالية بمجرد فتح التطبيق على المتصفح:

1. انتقل إلى **صفحة الإعدادات** في لوحة التحكم (التبويب الأخير بالأسفل).
2. **أدخل معرف جدول جوجل (Spreadsheet ID):** يمكنك نسخه مباشرة من رابط الـ Google Sheet في المتصفح الخاص بك (الرموز الموجودة في الرابط بعد `/d/` وقبل `/edit`).
3. **طريقتين سهلتين للربط:**
   * **الطريقة الأولى (مستحسنة وفورية):** اضغط على زر **"ربط حساب Google"** باللون الأبيض بالأعلى. سيظهر لك نافذة لتسجيل الدخول ببريدك الإلكتروني، بمجرد الموافقة ستأخذ لوحة التحكم صلاحية القراءة والكتابة والاتصال بملف الـ Sheets الخاص بك فوراً.
   * **الطريقة الثانية (حساب الخدمة):** إذا كنت تفضل استخدام مفتاح آمن ثابت، أدخل بريد حساب الخدمة (Google Service Account Email) والرمز السري (Private Key) الخاص بك في الخيارات المخصصة بالأسفل.
4. اضغط على **"حفظ معرف ورقة العمل"**.
5. الآن، سيقوم التطبيق تلقائياً وسلسة في كل دخول بسحب البيانات (Auto-Pull)، كما يمكنك استيراد البيانات يدوياً بضغطة زر **"تحميل واستيراد من Google Sheets"**، أو ترحيل وحفظ تعديلاتك محلياً بضغطة زر **"تصدير وترحيل إلى Google Sheets"**.

## 💾 نسخة احتياطية محلية للبيانات

* للأمان الإضافي، يتم حفظ نسخة مطابقة ومزامنة لكل البيانات على قرصك الصلب فوراً داخل ملف:
  📁 `data/db.json`
* يمكنك دائماً عمل نسخ مسودة لملف `db.json` وحفظه في فلاش ميموري (USB Drive) لضمان عدم ضياع أي بيان في المستقبل.
